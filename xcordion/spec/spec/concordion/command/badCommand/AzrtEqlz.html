Mispelled or unknown command

If you mistype a command in your document, for example concordion:azrtEqlz it will be highlighted as an unknwon command and cause a failure.

Example

Given this instrumented snippet:

<span concordion:azrtEqlz="#user.firstName">Fred</span>

Concordion will fail the to execute the test with a java.lang.RuntimeException with message of No such command 'azrtEqlz'
